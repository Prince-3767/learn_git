Setup Instructions

Follow these steps to set Life Journey as your Chrome new tab page:

Method 1: Using a Chrome Extension (Recommended)

Step 1: Install a New Tab Extension

  1. Open Google Chrome
  2. Go to the Chrome Web Store
  3. Search for "New Tab Redirect" or "Custom New Tab URL"
  4. Install one of these extensions (recommended: "New Tab Redirect")

Step 2: Host Your Life Journey Page

  1. Save all the Life Journey files (index.html, styles.css, script.js) in a folder
  2. Use one of these hosting options:
    • GitHub Pages (Free): Upload files to a GitHub repository and enable Pages
    • Netlify (Free): Drag and drop your folder to netlify.com
    • Local file: Use file:///path/to/your/index.html (may have limitations)

Step 3: Configure the Extension

  1. Click on the extension icon in Chrome
  2. Enter your hosted URL (e.g., https://yourusername.github.io/life-journey)
  3. Save the settings
  4. Open a new tab to test
Pro Tip: GitHub Pages is free and reliable. Create a repository named "life-journey", upload your files, and your page will be available at https://yourusername.github.io/life-journey

Method 2: Using Local Files (Advanced)

For Local Development:

  1. Save all files in a local folder
  2. Open Chrome and type chrome://extensions/
  3. Enable "Developer mode" (top-right toggle)
  4. Use a new tab extension and point it to your local file:// path
Note: Local files may have limited functionality due to browser security restrictions. Online hosting is recommended for the best experience.

Alternative Extensions to Try:

  • New Tab Redirect - Simple and reliable
  • Custom New Tab URL - Lightweight option
  • Momentum - Can be configured with custom URLs
  • Tabliss - Highly customizable

Troubleshooting

Page Not Loading?

  • Check if your URL is correct and accessible
  • Ensure HTTPS is used (not HTTP) for hosted pages
  • Try opening the URL directly in a browser first

Data Not Saving?

  • Make sure you're using HTTPS (localStorage restrictions)
  • Check if browser has localStorage enabled
  • Try refreshing the page and re-entering your birth date

Extension Not Working?

  • Disable and re-enable the extension
  • Try a different new tab extension
  • Check Chrome extension permissions
Quick Test: You can test the page immediately by opening index.html directly in Chrome before setting it as your new tab page.

Recommended Hosting: GitHub Pages Setup

  1. Create a GitHub account if you don't have one
  2. Create a new repository called "life-journey"
  3. Upload index.html, styles.css, and script.js
  4. Go to repository Settings → Pages
  5. Select "Deploy from a branch" and choose "main"
  6. Your page will be available at https://yourusername.github.io/life-journey
  7. Use this URL in your new tab extension
